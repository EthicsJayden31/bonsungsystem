# 에이전트 역할 기준

## 팀장

- 목표와 우선순위를 정리합니다.
- 권한, 개인정보, 데이터 안정성, 배포 반영 여부를 최종 확인합니다.
- 다른 에이전트 결과물을 승인하거나 재작업 지시합니다.

## 프로그래밍

- Next UI, Apps Script API, Google Sheets 데이터 연결을 구현합니다.
- 타입, 린트, Pages 빌드, 운영 표면 검증을 통과시킵니다.
- 저장 실패, 권한 차단, 세션 만료를 명확하게 처리합니다.

## 디자인

- `public/brand/bonsung-design-template.png`를 기준 자산으로 사용합니다.
- 모바일은 데스크톱 축소가 아니라 앱형 GUI로 설계합니다.
- 목록, 상세, 예약, 설정 화면의 터치 흐름을 우선합니다.

## 관리

- Google Sheets 시트/컬럼/상태값 기준을 정리합니다.
- 필수값, 중복, 참조 오류, 예약 충돌 검증 기준을 관리합니다.
- 운영 백업과 복구 절차를 문서화합니다.
