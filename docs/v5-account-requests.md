# 계정 요청·로그인 설정 v5

## 로그인 전 화면

- 로그인 제목은 `본성 통합 관리 시스템`으로 표시한다.
- admin과 원장(대표)은 `시스템 설정`에서 오른쪽 소개 제목·설명과 운영 공지 팝업을 수정한다.
- 공개 설정은 `설정` 시트에 저장되며 로그인하지 않은 사용자에게도 안전한 항목만 반환한다.

## 신규 계정 요청

1. 로그인 화면에서 `신규 계정 요청`을 누른다.
2. 이름, 희망 아이디, 계정 유형, 연락처, 이메일과 비밀번호를 제출한다.
3. 요청은 `계정요청` 시트에 `대기` 상태로 저장된다.
4. admin 또는 원장(대표)이 계정 유형, 직급, 연결 수강생을 확인하고 승인하거나 반려한다.
5. 승인된 사용자는 요청한 아이디와 비밀번호로 로그인한 뒤 비밀번호를 변경한다.

공개 요청에서는 admin 계정을 신청할 수 없다. 요청 비밀번호는 salt 해시로만 저장하며 승인 또는 반려 처리 후 요청 행에서 제거한다.

## admin 계정

- 기존 `시스템 관리자` 또는 `원장 관리자` 표시 이름은 스키마 v5 적용 시 한 번만 `admin`으로 변경한다.
- admin은 `환경 설정`에서 표시 이름, 이메일, 연락처와 비밀번호를 직접 변경할 수 있다.
- admin은 사람 계정이 아닌 시스템 관리 계정으로 유지되며 회의 참석자와 직원 명단에서 제외한다.
