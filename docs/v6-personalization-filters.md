# 개인화와 빠른 조회 v6

## 홈 화면 개인화

- `환경 설정`에서 로그인 후 시작 화면을 선택할 수 있다.
- 선택한 시작 화면이 권한상 접근 불가능하면 기본 홈 화면으로 이동한다.
- `화면 밀도`는 기본/촘촘하게 중 선택한다.
- 관리자 홈과 내 수업 홈은 표시할 카드 묶음을 직접 켜고 끌 수 있다.

## 빠른 조회

- `수강생` 목록은 이름, 연락처, 목표 검색과 상태, 담당 강사, 정렬 필터를 제공한다.
- `수업일지`는 내용 검색, 수강생, 강사, 수업 종류, 기간, 정렬 필터를 제공한다.
- `회의`는 제목, 안건, 장소 검색과 참석자, 상태, 기간, 정렬 필터를 제공한다.
- `일정`은 일정명, 수강생, 강사, 장소 검색과 빠른 날짜순/늦은 날짜순 정렬을 제공한다.

## 운영 데이터

- `계정` 시트에 `default_page`, `ui_density`, `dashboard_prefs_json` 필드를 추가한다.
- 새 계정, 계정 요청 승인, 초기 admin 생성 시 위 개인화 필드를 기본값으로 생성한다.
- 기존 운영 배포는 Apps Script 웹앱 버전 7부터 개인화 필드를 반환한다.

## 검증

- Chrome에서 `test.html?page=students`, `test.html?page=lesson-logs`, `test.html?page=meetings`, `test.html?page=calendar`, `test.html?page=profile`을 확인한다.
- 모바일 폭에서는 필터 입력과 선택지가 2열 또는 1열로 접히며 가로 스크롤 없이 세로 스크롤로 사용한다.
